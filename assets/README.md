# Assets Folder

This folder contains all asset files for the portfolio site.

## Structure

```
assets/
├── pictures/     # Image files (jpg, png, svg, etc.)
└── README.md     # This file
```

## Usage

Reference assets in your HTML/CSS using:
- `../assets/pictures/filename.jpg` (from projects folder)
- `assets/pictures/filename.jpg` (from root)

## Image Guidelines

- Use descriptive filenames
- Optimize images for web (compress before uploading)
- Recommended formats: JPG for photos, PNG for graphics with transparency, SVG for icons
